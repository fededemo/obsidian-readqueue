Configuración de la carpeta de la vault

Cargando…

  1. Tocá "Elegir carpeta de la vault".
  2. Navegá a …/fedenotes/Inbox/Kindle y seleccionala.
  3. En el cartel de Chrome, elegí "Edit files" / "Editar archivos" (y "on every visit" si aparece).
  4. Cuando el recuadro de arriba diga ✅ permiso OK, volvé al popup de la extensión y tocá "Sincronizar ahora".

Esta pestaña es una página completa (no el popup) a propósito: el permiso de la File System Access API se otorga y persiste de forma confiable acá, cosa que en el popup se rompe cuando se abre el diálogo del sistema.
